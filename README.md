XièXie Compiler and VirtualMachine (Version 2)
==============================================

Features
--------

- High Level Programming Language
- Compiler written in C++11 (in progress)
- Virtual Assembler (XASM)
- Virtual Machine (written in C99, with wrapper written in C++11)
